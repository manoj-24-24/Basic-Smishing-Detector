# 📱 Smishing Detection Dashboard

A simple cybersecurity dashboard that detects **smishing (SMS phishing)** messages and classifies them as **Secure, Suspicious, or Fraudulent**.

🌍 **Live Demo:**  
https://manoj-24-24.github.io/Basic-Smishing-Detector/

---

## 🛡️ Features

- SMS Smishing Detection  
- Fraud Risk Classification  
- Suspicious URL Detection  
- Neon Cybersecurity Dashboard UI  
- Risk Visualization Graph  
- Cybersecurity Awareness News Panel  

---

## ⚙️ Technologies Used

- HTML  
- CSS (Neon UI Design)  
- JavaScript  
- Chart.js  
- GitHub Pages
